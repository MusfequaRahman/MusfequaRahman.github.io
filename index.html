Publications

CAMFusion: Context-Aware Multi-Modal Fusion Framework for Detecting Sarcasm and Humor Integrating Video and Textual Cues
Musfequa Rahman, M.A.M. Provath, K. Deb, P.K. Dhar, T. Shimamura
IEEE Access, 2025
This work introduces a Context-Aware Multi-modal Fusion Framework for detecting sarcasm and humor in Bangla videos. A new multimodal dataset was curated, annotated across humor, sarcasm, and normal categories. The framework fuses visual features from an Efficient Fusion Time-Distributed MobileNetV2 with textual representations from BiLSTM and GRU, attaining 90% accuracy. A novel Bangla Text Extraction Algorithm enhances contextual retrieval from complex frames, with cross-dataset evaluations confirming the model’s robustness and generalizability.
DeepGuard: Enhancing Violence Detection in Smart Cities through Deep Learning
M.A.M. Provath, Musfequa Rahman, K. Deb, P.K. Dhar, T. Shimamura
IEEE Access, 2025
To advance research on video-based security systems, this work introduces a specialized dataset on political violence, comprising 480 labeled clips across four categories. Building on this resource, we fine-tuned MoViNet-A0, attaining 92.86% accuracy with only 1.904M parameters. A custom keyframe extraction algorithm leveraging temporal and pixel-level cues further enhances stability and detection accuracy. Extensive evaluations demonstrate clear improvements over existing approaches, establishing a strong benchmark for future work in violence detection for smart cities.
ADBNet: an attention-guided deep broad convolutional neural network for the classification of breast cancer histopathology images
Musfequa Rahman, K. Deb, P.K. Dhar, T. Shimamura
IEEE Access, 2024
ADBNet, an attention-guided deep broad convolutional network for breast cancer histopathology classification. By combining feature-selective attention, magnification-resilient design, and GAN–diffusion-based data augmentation, ADBNet achieves up to 99.52% accuracy, surpassing existing methods and offering a robust tool for computer-aided diagnosis.
CricActionNet: A Comprehensive Framework for Cricket Action Recognition Using Keyframe Extraction and DenseNet-Transformer Based Architecture
TM Mitul, Musfequa Rahman, K. Deb
Engineering Science and Technology, an International Journal, 2025 (In Submission)
CricShot14, the largest dataset for cricket action recognition, spanning fourteen distinct action categories. A novel keyframe extraction technique tailored to cricket videos was developed, enabling more precise action analysis. By integrating DenseNet121 for feature extraction with a Transformer for sequence modeling, our framework achieves 93.44% accuracy. Extensive evaluations on CricShot10 and CKT datasets confirm superior performance over existing methods, establishing a new benchmark for Human Action Recognition in sports.
Efficient Bangla Multimodal Humor Detection with Lightweight Dictionary-Based Transformations
Musfequa Rahman, M.A.M. Provath, K. Deb
3rd ICMAM, 2024
The Lite-MDETR framework leverages Dictionary-Lookup Transformations (DLT) to cut memory usage by 52% and reduce parameters from 100M to 48M, while still achieving 86% accuracy in Bangla multimodal humor detection. A new dataset capturing diverse visual–verbal contexts further strengthens scalability and resource efficiency.
Classifying Breast Cancer Using Deep Convolutional Neural Network Method
Musfequa Rahman, K. Deb, K.H. Jo
International Workshop on Frontiers of Computer Vision (Springer Nature Singapore), 2023
The Breast Cancer CNN Classifier uses deep learning on the BreaKHis dataset to analyze histopathology images, reaching 99.5% binary accuracy and 96.1% multiclass accuracy, without the need for manual feature engineering.
